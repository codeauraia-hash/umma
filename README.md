# UMMA ACTIVE - Proyecto Web Organizado

## Estructura del Proyecto

Este proyecto ha sido organizado con una estructura modular y separada para facilitar el mantenimiento y escalabilidad.

### 📁 Estructura de Carpetas

```
umma/
├── css/
│   └── style.css          # Estilos CSS del sitio
├── js/
│   └── script.js          # Funcionalidad JavaScript
├── assets/
│   └── favicon.ico        # Icono del sitio
├── images/               # Imágenes y recursos visuales
├── index.html            # Página principal
└── README.md             # Documentación del proyecto
```

### 🎨 Características

- **Diseño Responsivo**: Adaptado para móviles, tablets y escritorios
- **E-commerce Completo**: Catálogo de productos, carrito de compras y checkout
- **Interfaz Moderna**: Diseño oscuro con acentos púrpura
- **Animaciones Suaves**: Transiciones y efectos visuales atractivos
- **Navegación Intuitiva**: Menú principal y navegación por categorías

### 🛒 Funcionalidades

- **Catálogo de Productos**: 
  - Calzas deportivas
  - Corpiños deportivos  
  - Tops deportivos
- **Sistema de Carrito**: Agregar, eliminar y modificar cantidades
- **Checkout por Email**: Procesamiento de pedidos vía correo electrónico
- **Integración WhatsApp**: Contacto directo con clientes
- **Filtros por Categoría**: Navegación fácil entre productos

### 🚀 Tecnologías Utilizadas

- **HTML5**: Estructura semántica moderna
- **CSS3**: Diseño con variables CSS, Grid y Flexbox
- **JavaScript ES6+**: Funcionalidad interactiva y dinámica
- **Google Fonts**: Tipografía Montserrat
- **Diseño Mobile-First**: Responsive design prioritizando móviles

### 📦 Organización del Código

- **Separación de Responsabilidades**: CSS, JavaScript y HTML en archivos separados
- **Código Modular**: Funciones reutilizables y bien organizadas
- **Comentarios Descriptivos**: Código documentado para fácil mantenimiento
- **Buenas Prácticas**: Estructura limpia y optimizada

### 🎯 Objetivo del Proyecto

UMMA ACTIVE es una tienda online de ropa deportiva diseñada para ofrecer una experiencia de compra moderna y fluida, con foco en la usabilidad y el rendimiento.
